1. Выведите содержимое fstab. Что хранится в fstab?
-- cat /etc/fstab
2. Добавьте в виртуальную машину ещё один диск
-- в главном окне VirtualBox в «настройки», там раздел «Хранилище», в меню «Хранилище» выбрать «Контроллер: SATA», а затем нажать «Добавить диск» и «Добавить жёсткий диск», во всплывающем окне «Создать новый диск» и далее по инструкциям мастера создания виртуального жёсткого диска 
3. Узнайте как система видит ваш диск - выведите информацию о блочных устройствах
-- lsblk
4. С помощью полученной информации создайте на диске таблицу разделов и фаловую систему ext4
-- mkfs.ext4 /dev/sda
5. Примонитруте диск в каталог /mnt
-- mount /dev/sda /mnt
6. Зайдите в каталог и создайте там файл
-- touch /mnt/oigear
7. Отмонтируйте диск и проверье остались ли файлы
-- sudo umount /mnt/sda
-- файлы "заморозятся" и будут доступны при перемонтировании
8. Сделайте так чтобы диск автоматически подключался при загрузке систем (добавьте информацию о нём с fstab)
-- blkid (узнаём UUID диска)
-- nano /etc/fstab
-- пишем UUID={UUID} /dev/sda ext4 default 0 0
Проверьте корретность записанных в fstab данных перед перезагрузкой
-- cat etc/fstab
Перезагрущите систему и убедитесь что диск был подключён к системе
-- всё работает

1. Raid массивы, что такое и какие бывают	
-- Redundant Array of Independent Disks; оно объединяет несколько дисков в один с увеличенным объемом и скоростью передачи данных. Существует несколько уровней RAID: например RAID 0 с объединением объёмов и скоростей записи, RAID 1 для "дублирования" файлов с объёмом RAID как минимальный объём из объединённых дисков 
2. Добавьте в виртуальную машину 2 диска отформатируйте их в ext4
-- mkfs.ext4 /dev/sda
-- mkfs.ext4 /dev/sda1
3. Создайте из них raid 0 массив
-- mdadm --create /dev/md0 --level=stripe --raid-devices=2 /dev/sda /dev/sda1
4. Проверье всё ли работает
-- cat /proc/mdstat
5. Удалите raid0 и создайте raid1
-- mdadm --stop /dev/md0
-- mdadm --create /dev/md1 --level=mirror --raid-devices=2 /dev/sdc1 /dev/sdd1
6. В чём между ними разница?
-- разница в том, что RAID 0 работает с объединением объёмов и скоростей записи, RAID 1 работает с объёмом RAID как минимальный объём из объединённых дисков при этом с зеркальным копированием файлов для большей безопасности
7. Есть ли файловые системы которые поддерживают raid массивы без стороненго ПО
-- да, например ZFS
8. Можно ли создать raid массив во время установки системы?
-- да, можно задать параметры для создания RAID-массива из нескольких жестких дисков при установке системы
