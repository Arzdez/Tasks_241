Работаем в консольльльльльльке
1. Переместиться между директориями -- cd <path>(.. чтобы подняться выше)
2. Вывести список файлов в директории -- ls
3. Вывести вообще все -- -la
4. Создать папку с подпапками -- mkdir -p q/w/e
5. Внутри папки создать файлик и записать в него что-нибудь -- touch f.txt && echo "smth" >> f.txt
6. Переместить файл из одно директории в другую -- mv <file> <path>
7. Скопировать файл из одной директории в другую -- cp <file> <path>
8. Переименовать файл -- mv <file> <file w/ a new name>
9. Сравнить содержимое файлов --  diff <option(-s для совпадений)> <file1> <file2>
10. Отсортировать содержимое файлы -- sort <option(-r for reverse, )> <file>
11. Удалить все файлы и папки -- rm -r <path>
